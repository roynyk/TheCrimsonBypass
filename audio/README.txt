Taruh semua file efek suara (.mp3, .wav, atau .ogg) Anda di dalam folder 'audio' ini.

Contoh file yang direkomendasikan untuk game "THE CRIMSON BYPASS":
1. ambient_night.mp3 (Suara angin malam sunyi untuk musik latar berulang/loop)
2. motorcycle_roar.mp3 (Suara motor lewat berkecepatan tinggi)
3. metal_strike.mp3 (Suara benturan pipa besi/tabrakan)
4. heartbeat.mp3 (Suara detak jantung yang semakin cepat saat bersembunyi)

Anda dapat memanggil suara-suara ini di dalam cerita Twine menggunakan kode JavaScript sederhana yang telah saya siapkan.
